# Images Required for SignEase App

Please add the following images to the `assets/images/` folder:

## Splash Screen
- `splash-hands.png` - Image of 6 hand gestures in a horizontal row (white hands)
- `splash-arrow.png` - Orange curved arrow underneath the hands

## Onboarding Screens
- `onboarding-1-illustration.png` - Illustration showing two people communicating via sign language (woman on left, man in video call frame on right)
- `onboarding-2-heart.png` - Abstract illustration of hands forming a heart shape with red string

## Home Screen
- `home-banner.png` - Hero banner illustration with ear icon, two people, and mute symbol
- `feature-translator.png` - Icon for Sign Language Translator feature
- `feature-learning.png` - Icon for Learning Modules feature  
- `feature-ai-tutor.png` - Icon for AI Tutor feature (3D cartoon character)
- `feature-progress.png` - Icon for Practice & Progress feature (bar chart with stars)

## Image Specifications
- All images should be PNG format with transparent backgrounds where applicable
- Recommended sizes:
  - Splash images: ~300x100px (hands), ~200x40px (arrow)
  - Onboarding illustrations: ~400x300px
  - Home banner: ~150x150px
  - Feature cards: ~120x120px

All images are referenced in the screen components and will display once added to this folder.
